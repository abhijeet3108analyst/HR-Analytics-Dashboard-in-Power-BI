# 👨‍💼 Employee Performance & Salary Analysis Dashboard

## 📌 Project Overview

The Employee Performance & Salary Analysis Dashboard is an interactive Power BI solution designed to help HR teams and management analyze workforce performance, compensation trends, training effectiveness, and employee contribution.

This dashboard transforms employee data into actionable insights, enabling organizations to make informed decisions regarding talent management, salary planning, promotions, and employee development.

---

## 🎯 Business Problem

Organizations often struggle to answer critical HR questions such as:

- Which departments have the highest-performing employees?
- Does employee training improve performance?
- Is compensation aligned with employee contribution?
- Are there salary disparities across departments or genders?
- Which employees have the highest impact on organizational performance?

This dashboard addresses these challenges through data-driven analysis and interactive visualizations.

## 1️⃣ Executive Overview Dashboard

### Purpose
Provides a high-level summary of workforce performance and compensation.

### Key KPIs
- Total Employees
- Average Salary
- Average Performance Rating
- Promotion Rate %

### Visualizations
- Employees by Department
- Salary by Department
- Performance Distribution
- Department Filters
- Gender Filters
- Location Filters

### Business Insights
- Understand workforce distribution.
- Monitor overall employee performance.
- Evaluate department-level salary allocation.
- Track promotion trends.

## 2️⃣ Performance Analysis Dashboard
